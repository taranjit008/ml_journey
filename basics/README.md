# Basics

This folder contains my practice code while learning fundamentals of python libraries used in data analysis and machine learning.

## Contents

- **NumPy**- Arrays, indexing, broadcasting, and mathematical operations.
- **Pandas**- Data loading, cleaning, filtering, grouping, and analysis.
- **Matplotlib**- Data visualization using line, bar, scatter, and other plots.

The code in this  folder focuses on understanding core concepts before applying them in complete projects.
